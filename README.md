# Doxira – AI Powered Document Editor 📄✨

<img width="1488" height="720" alt="Cover page 2 (1)" src="https://github.com/user-attachments/assets/9abc182e-37fb-4db2-8d2e-966fa175e177" />


Doxira is an intelligent document editor designed to help users create, edit, and enhance professional documents such as CVs, reports, and business files using AI assistance directly inside the document.

The goal of Doxira is simple: **write better documents, faster, with AI that understands context.**

---

## 🚀 Features

- **AI-Powered Document Editing**
  - Real-time content generation, rewriting, and enhancement
  - Grammar correction and intelligent phrasing suggestions
  - Context-aware recommendations based on the current document

- **In-Document AI Chat**
  - Built-in AI sidebar for step-by-step guidance
  - Ask questions, refine sections, and improve structure without leaving the editor

- **Multi-Format Support**
  - Read and parse documents from PDF, DOCX, PPT, and Excel
  - Export documents to **PDF** and **Word**

- **Templates & Productivity Tools**
  - Ready-to-use templates for CVs, reports, and business documents
  - Structured writing assistance for professional outputs

- **Cloud Storage & File Management**
  - Secure document upload and storage
  - Organize and manage files efficiently

- **Powerful Search System**
  - Fast document search across cloud storage and local device files
  - Indexed for quick discovery

- **Subscription & Payments**
  - Subscription-based access control
  - Integrated payment gateways for premium features

- **Modern UI / UX**
  - Clean, productivity-focused design
  - Dark mode and multiple themes
  - Smooth and responsive user experience

---

## 🧠 AI & Technologies

- Large Language Models (LLMs)
- LangChain for context-aware AI workflows
- Intelligent document parsing and understanding
- Secure backend for file handling and access control
